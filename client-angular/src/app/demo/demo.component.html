App Demo

A walkthrough of how the app works.

Arrival

This is the first thing you see when starting. You will find pages such as Customers, Pricing, Demo, New Ticket, and View Tickets.

Home
Ticket Creation

Upon clicking on New Ticket you will come to a screen allowing the user to submit a issue ticket. The user fills out the "Created By" form along with a description of the issue.

Submit

Viewing Tickets

This is where the user can look at existing tickets. The user may then click on a ticket and edit or update the information.

View Tickets
Edit Ticket

After selecting a ticket the user is brought here where they can edit the information regarding the selected ticket.
